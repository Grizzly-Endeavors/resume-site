# The Factory (Simulation Game)
**Dates:** Ongoing
**Skills:** Systems Engineering, Game Logic, Optimization

Developed a sophisticated factory simulation in Rust using the Bevy ECS engine, featuring autonomous worker AI with BFS pathfinding and emergency displacement logic.
- Systems Engineering: Developed a sophisticated factory simulation in Rust using the Bevy ECS engine, featuring autonomous worker AI with BFS pathfinding and emergency displacement logic.
- Game Logic: Implemented complex systems for network infrastructure (power/compute grids), progressive map scanning, and modular building assembly using data-driven configuration (RON).
- Optimization: Designed dual-layer network models and priority-based task queuing systems to manage thousands of entities efficiently.
