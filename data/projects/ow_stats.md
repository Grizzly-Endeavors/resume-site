# OW-Stats (Analytics Platform)
**Dates:** Ongoing
**Skills:** AI Integration, Data Engineering, Visualization

Integrated Google Gemini Vision AI to parse unstructured game screenshots into structured Pydantic models with automated JSON schema generation.
- AI Integration: Integrated Google Gemini Vision AI to parse unstructured game screenshots into structured Pydantic models with automated JSON schema generation.
- Data Engineering: Designed a normalized SQLite database schema with an object-oriented data access layer to execute complex statistical queries (win rates by role, map, hero).
- Visualization: Built an interactive frontend using Streamlit and Altair, featuring dynamic filtering and real-time data visualization of competitive performance metrics.
